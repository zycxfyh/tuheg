## 🏗️ 技术架构亮点

### 多层次架构
```
┌─────────────────┐
│   前端应用      │  Vue 3 + TypeScript
├─────────────────┤
│   API网关       │  NestJS + WebSocket
├─────────────────┤
│   AI代理层      │  Creation/Logic/Narrative Agent
├─────────────────┤
│ VCPToolBox核心  │  6个核心模块 + 5个插件
├─────────────────┤
│   数据层        │  PostgreSQL + Redis + 文件存储
└─────────────────┘
```

### 插件生态系统
- **6种协议类型**: static, messagePreprocessor, synchronous, asynchronous, service, dynamic
- **标准化接口**: VCP协议保证兼容性
- **热插拔机制**: 运行时动态加载
- **版本管理**: 插件生命周期管理
