## 🎯 项目概述

### 📋 项目简介

"创世星环"是一个基于AI驱动的叙事游戏创作平台，采用微服务架构设计，为用户提供沉浸式的AI辅助游戏创作体验。

### 🏗️ 核心架构

- **微服务架构**: 独立部署的AI代理服务
- **事件驱动设计**: RabbitMQ消息队列通信
- **AI驱动核心**: LangChain + 多种AI模型集成
- **现代化技术栈**: Vue 3 + NestJS + TypeScript + Prisma

### 🎮 核心功能

- 🤖 **AI叙事生成**: 智能的故事线和角色发展
- 🎨 **创意世界构建**: AI辅助的世界观设计
- 👥 **多角色协作**: 支持多人协作创作
- 📊 **实时游戏体验**: WebSocket实时交互
- 🔒 **企业级安全**: 多层安全防护体系
