## 📋 项目总览

<div align="center">

> **创世星环 (Creation Ring)** 是一个基于微服务架构的AI驱动交互式叙事游戏生成系统，采用多Agent协作技术，为用户提供沉浸式的创作体验。

</div>

### 🎯 项目愿景

构建一个**AI创作操作系统的操作系统**，通过四个专门的AI代理（Creation、Logic、Narrative、Backend Gateway）的智能协作，让AI成为创作的真正伙伴，从概念生成到完整世界构建的全流程AI赋能。

### 核心技术栈

#### 🎨 前端技术栈
- **框架**: Vue 3 (Composition API) + TypeScript 5.0+
- **构建**: Vite 4.x + ESBuild
- **状态管理**: Pinia 2.x
- **路由**: Vue Router 4.x
- **测试**: Vitest + Vue Test Utils + Playwright
- **样式**: CSS Modules + PostCSS

#### ⚡ 后端技术栈
- **框架**: NestJS 10.x + TypeScript 5.0+
- **数据库**: PostgreSQL 15 + Prisma ORM
- **缓存**: Redis 7.x (多层缓存策略)
- **消息队列**: RabbitMQ 3.x (事件驱动架构)
- **API文档**: OpenAPI 3.0 + Swagger UI
- **测试**: Jest + Supertest

#### 🤖 AI集成技术栈
- **Agent架构**: 4个专用AI代理服务
- **通信协议**: RabbitMQ + WebSocket
- **状态管理**: Redis缓存 + PostgreSQL持久化
- **错误处理**: 死信队列 + 重试机制
- **监控**: 性能指标 + 健康检查

#### 🏭 基础设施技术栈
- **容器化**: Docker + Docker Compose
- **编排**: Kubernetes (生产环境)
- **监控**: Prometheus + Grafana + AlertManager
- **日志**: 结构化日志 + ELK Stack
- **CI/CD**: GitHub Actions + 自动化部署

#### 🔒 安全技术栈
- **认证**: JWT + Passport.js
- **授权**: RBAC + 自定义中间件
- **加密**: AES-256 + 环境变量管理
- **审计**: 操作日志 + 安全事件追踪
- **合规**: GDPR准备 + 数据保护
