## 🤝 贡献指南

### 代码贡献

我们欢迎社区贡献！请遵循以下步骤：

1. **Fork项目**
   ```bash
   git clone https://github.com/creation-ring/vcptoolbox-sdk.git
   cd vcptoolbox-sdk
   ```

2. **创建功能分支**
   ```bash
   git checkout -b feature/AmazingFeature
   ```

3. **提交更改**
   ```bash
   git commit -m 'Add some AmazingFeature'
   ```

4. **推送分支**
   ```bash
   git push origin feature/AmazingFeature
   ```

5. **创建Pull Request**
   - 使用PR模板
   - 详细描述更改内容
   - 关联相关Issues

### 贡献类型

- 🐛 **Bug修复** - 修复现有bug
- ✨ **新功能** - 添加新功能
- 📚 **文档** - 改进文档
- 🧪 **测试** - 添加或改进测试
- 🎨 **样式** - 改进UI/UX
- ♻️ **重构** - 代码重构
- 🚀 **性能** - 性能优化

### 行为准则

我们致力于维护一个友好、包容的社区。请遵守：
